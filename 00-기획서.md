## 1. 채널 정체성

### 한 줄 정의
**테슬라 주주를 위한, 노이즈 없는 일일 브리핑.**

### 타겟 페르소나
- **테슬라 주주·예비 주주·장기 보유자.** 단타 트레이더 아님. 분기·연 단위로 회사를 본다.
- 영어 1차 자료(SEC 8-K, 어닝 콜, 머스크 X 게시물, NHTSA 문서)에 직접 접근 가능하지만 시간이 없다.
- 트위터/유튜브에서 흩어진 정보를 모아서 보고 싶지만 "TSLA 떡상" 식 클릭베이트는 거부감.

### 톤 & 매너
- **냉정한 친절함.** 데이터를 먼저, 해석은 분리해서, 추측은 추측이라고 명시.
- 머스크 발언은 "발언 자체"와 "회사 영향" 두 층위로 분리해 다룬다.
- "사야 한다 / 팔아야 한다" 말하지 않음. 의사결정 재료만 제공.

### 차별점
- 모든 콘텐츠에 **출처 카운트**(SEC 문서 / 1차 자료 / 외신 / 추측)를 카드 메타에 노출 — 허라비 채널의 "검증 도장" 시스템을 테슬라용으로 재해석.
- 영상·뉴스레터·랜딩 한 곳에서 같은 4개 카테고리로 정렬.

## 2. 사이트 구조 (랜딩 1페이지 원칙)

이번 빌드의 핵심 결정: **카테고리 전용 페이지 없음.** 랜딩 한 페이지에서 모든 정보를 본다. 추후 트래픽 데이터를 보고 카테고리 페이지 분리 여부를 결정한다.

### 섹션 순서 (위→아래)
1. **상단 nav** — 로고(마스코트 + "TSLA Brief") · 카테고리 4개 앵커 · 구독 CTA
2. **히어로 + 뉴스레터 구독** — "노이즈 없는 테슬라 브리핑" 한 줄 + 이메일 입력 폼 + 발송 주기(평일 아침 7시) + 누적 발행 회차
3. **주가·시총 지표 위젯** — TSLA 현재가 / 시가총액 / 52주 고저 / P/E / 다음 어닝일 (정적 더미 → 추후 API)
4. **4개 카테고리 최신 이슈 카드** — 주가·실적 / 신차·제품 / FSD / 머스크 각 1장씩 하이라이트
5. **유튜브 최신 영상 그리드** — 6개 썸네일 카드 (3×2 또는 2×3)
6. **푸터** — 출처 정책, 면책고지, 마스코트, 구독 채널 링크

### 의도적으로 뺀 것
- 풀 카테고리 페이지 (카드에서 외부 콘텐츠로 직링크)
- 검색·필터 (1페이지 정보량으로 불필요)
- 광고·제휴 영역 (트러스트 우선)

## 3. 4개 카테고리 정의

| 카테고리 | 다루는 것 | 다루지 않는 것 | 카드 컬러 액센트 |
|---|---|---|---|
| **주가·실적** | 분기 실적, 가이던스, 컨센서스, 거래량 이상치, 옵션 흐름 | 일일 시세 코멘트 | 레드 (`--tesla-red`) |
| **신차·제품** | 모델 라인업, 사이버트럭, Optimus, 로보택시, 신차 가격·인도량 | 차주 리뷰, 시승기 | 일렉트릭 블루 (`--electric`) |
| **FSD** | 버전 릴리스, 인터벤션 데이터, NHTSA 조사, 로보택시 운영 | 일반인 시승 영상 | 시안 (`--cyan`) |
| **머스크·일론** | 보상안, X 게시물 중 회사 영향, 정치·정책 코멘트의 회사 리스크 | 개인사·연애 | 앰버 (`--amber`) |

각 카테고리 카드 우상단에 작은 컬러 도트 + 카테고리명 — 가독성과 분류 일관성.

## 4. 비주얼 시스템

### 컬러 (다크 모드 기본)
```css
/* 배경 위계 */
--bg:        #0A0A0B;   /* 페이지 베이스, 거의 검정 */
--bg-2:      #111114;   /* 카드 배경 */
--bg-3:      #181820;   /* 호버·강조 카드 */
--line:      #2A2A33;
--line-mute: #1E1E26;

/* 텍스트 위계 */
--ink:       #F4F4F5;
--ink-mid:   #B8B8C0;
--ink-mute:  #707078;

/* 브랜드 액센트 */
--tesla-red: #E31937;   /* 메인 액센트, CTA, 강조 */
--red-deep:  #A30E25;   /* 호버 */
--red-soft:  rgba(227, 25, 55, 0.12);

/* 카테고리 액센트 */
--electric:  #1B6CFF;   /* 신차·제품 */
--cyan:      #22D3EE;   /* FSD */
--amber:     #F59E0B;   /* 머스크 */

/* 데이터 시각화 */
--up:        #16A34A;   /* 상승 */
--down:      #EF4444;   /* 하락, tesla-red와 구분 */
```

### 타이포그래피
- **본문:** Pretendard Variable (한국어 + 영문 숫자)
- **숫자·티커:** IBM Plex Mono — 주가, %, 시총, 날짜에 monospace 적용 (테크·재무 신뢰감)
- **헤드라인 강조 italic:** Inter italic (허라비의 Cormorant 자리 — 더 테크 톤)
- **스케일:** h1 `clamp(40px, 6vw, 64px)` 800 · h2 `clamp(26px, 3vw, 36px)` 700 · body 16/1.65 · eyebrow 12/0.18em letter-spacing

### 카드·인터랙션 시그니처
- 카드 라운드 16px, border 1px solid `--line`, hover 시 `--bg-3` + border `--tesla-red` 미세 발광 (box-shadow red soft glow 24px)
- 카드 좌상단에 카테고리 도트 + 라벨(eyebrow), 우상단에 시간 (`14h ago`)
- 거대 워터마크 숫자 대신 **티커·분기 라벨**을 monospace로: `TSLA · 2026.Q1`, `FSD v13.4`
- CTA 화살표: `→` hover 시 `gap: 4px → 10px` 분리 (허라비 패턴 차용)

### 마스코트 (신규 디자인 방향)
**이름 후보:** Volty (볼티) / Sparky (스파키) / Rho (로) — 회의 후 결정.

**컨셉:** 작은 차량 실루엣을 추상화한 **둥근 캐릭터**. 머리는 사이버트럭 프로파일, 눈은 옵티머스 LED 라인. 친근하되 너무 귀엽지 않게 — 주주 톤과 충돌 방지.

**노출 원칙 (허라비와 동일):**
1. nav 로고 (28×28)
2. footer 브랜드 (80×80)
3. "출처 검증 도장" SVG — 콘텐츠 신뢰도 표시 (이번 빌드에는 미포함, 추후 콘텐츠 페이지에서 사용)

이번 세션에서 마스코트 SVG 2종 후보 제작:
- **후보 A — 차량 실루엣 + 미니멀:** 사이버트럭 사이드뷰 추상화, 단색 라인 아트
- **후보 B — 봇 헤드 + 친근:** Optimus 헤드 추상화 + 작은 안테나, 둥근 형태

## 5. 영상 채널 컨셉 (간단 가이드)

### 영상 포맷
- **숏폼(60초 이내, 세로 1080×1920):** 1일 1편, 그날의 핵심 한 가지. 머스크 헤드라인 발언 / FSD 신버전 / 어닝 D-day 등
- **롱폼(8~15분, 가로 1920×1080):** 주 1편, 주간 정리 또는 분기 실적 해설

### 비주얼 스타일 (허라비 Style 차용 + 다크 변형)
- 다크 배경 (`#0A0A0B`) + 레드 액센트
- 인트로: 3줄 카드 스택 + 우상단 도장(`VERIFIED` 또는 `SOURCE.SEC`) + 마스코트
- 본문 비트: `POINT 01~N`, monospace 데이터 박스 (주가/시총/날짜 강조)
- 아웃트로: 마스코트 중앙 + 구독·알림 박스
- 자막: 검정 박스 + 흰 글씨 (허라비 ThemedSubtitle 그대로 차용)
- 보이스: `ko-KR-Chirp3-HD-*` 계열 — 너무 활기차지 않은 톤 (Orus 또는 더 차분한 목소리 후보)

### 카테고리별 시리즈 배지
- `STOCK` (레드)
- `PRODUCT` (블루)
- `FSD` (시안)
- `MUSK` (앰버)

영상 우상단 헤더에 마스코트 + 시리즈 배지 (허라비 TipScene과 동일 패턴).

- **분량:** 본문 약 800자, 스크롤 1.5회

5. **푸터** — 출처 정책 링크, 구독 해지

- 머스크 게시물은 원문 인용 → 회사 영향 한 줄.

## 7. 출처·검증 정책 (트러스트 핵심)

각 콘텐츠에 4단계 출처 태그를 단다:
- 🟢 **1차 자료** — SEC 8-K/10-Q/10-K, 어닝 콜 트랜스크립트, NHTSA 문서
- 🔵 **공식 발언** — Tesla IR, 머스크 X 게시물 (원문 링크)
- 🟡 **신뢰 외신** — Bloomberg, Reuters, WSJ, FT, Electrek
- 🟠 **추측·루머** — 명시적으로 라벨링, 본문에서 추측임을 반복 고지

랜딩 페이지 카드 메타에 출처 태그 도트 + 카운트 노출 (예: `🟢 2 · 🔵 1`).
